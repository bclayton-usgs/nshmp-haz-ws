nshmp-haz-ws provides the following services:
  • DeterministicSpectra
  • HazardCurve
See each page above for instructions on use.

A USGS hazard curve may be computed by supplying a slash-delimited
query consisting of year, model, intensity measure type (imt), longitude,
and latitude. For example:

  http://.../nshmp-haz-ws/HazardCurve/2008/WUS/PGA/-118.25/34.05

where:
  year = [2008, 2014]
  model = [WUS, CEUS]
  imt = see docs for options

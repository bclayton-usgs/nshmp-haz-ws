nshmp-haz-ws

Service: hazard

Compute probabilisitic seismic hazard curves at a site of interest. Supply a slash-delimited or name-value pair query consisting of:

  • edition [E2008, E2014]
  • region [COUS, WUS, CEUS]
  • longitude
  • latitude
  • imt (intensity measure type) [PGA, SA0P2, SA1P0]
  • vs30 [180, 259, 360, 537, 1150, 2000]
/hazard/E2008/WUS/-118.25/34.05/PGA/760
/hazard?edition=E2008&region=WUS&longitude=-118.25&latitude=34.05&imt=PGA&vs30=760

For both forms, multiple, comma-delimited IMTs may be supplied. For the slash delimited format, 'any' may be supplied as the IMT id and the service will return curves for all supported IMTs. For the name-value pair format, omit the IMT argument to return curves for all supported IMTs.

See usage for parameter dependencies.

Service: deagg

Deaggregate seismic hazard. Construct queries identical to those defined for hazard, above, with an additional parameter:

  • returnperiod (in years)
/deagg/E2008/WUS/-118.25/34.05/PGA/760/2475
/deagg?edition=E2008&region=WUS&longitude=-118.25&latitude=34.05&imt=PGA&vs30=760&returnperiod=2475

The deaggregation service only supports calculations for a single IMT, which must be specified.

See usage for parameter dependencies.

Service: spectra

Compute determinisitic reponse spectra. Supply a name-value pair query consisting of:

  • gmm (one or more)
  • mag
  • rjb
  • rrup
  • rx
  • dip
  • width
  • ztop
  • zhyp
  • rake
  • vs30
  • vsinf
  • z2p5
  • z1p0
/spectra?gmm=ZHAO_06_INTER&gmm=BCHYDRO_12_INTER&gmm=AM_09_INTER&mw=87.8&rjb=10.0&rrup=23.0&rx=32.0&dip=30.0&width=25.0&ztop=10.0&zhyp=20.0&rake=90.0&vs30=760.0&vsinf=true&z2p5=NaN&z1p0=NaN

Not all parameters are used by every grounf motion model (gmm). At least one 'gmm' must be specified. Default values will be used for any parameters not specified. will See usage for parameter dependencies.
